Konami Services Webpage
Project Overview

This project is a simple Konami inspired Services Webpage built using HTML and CSS. It showcases a modern layout using Flexbox and includes a visually appealing neon themed design to create a gaming style user interface.

The goal of this project is to demonstrate understanding of:

Web page structure using HTML
Layout design using Flexbox
Styling with CSS 
Version control using Git (with multiple commits)

Technologies Used
HTML
CSS (Inline styling + Flexbox)
Git & GitHub

 Features
 Navigation bar with links (Home, About, Services)
 Neon gradient background for a modern gaming look
Glowing text and UI elements
Flexbox layout
Responsive design
